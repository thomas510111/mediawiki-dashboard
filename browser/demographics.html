Attraction / retention

Attracted: Number of contributors entering the project for each cohort (age group). Each bar represents an age group which is defined by the time since they entered the project. This is sometimes also called "magnetism".

Retained: Number of contributors who are still active nowadays who entered the project in each cohort. This is sometimes also called "stickiness".

The ratio between retained and attracted contributors for a cohort represents the retention ratio for that cohort (fraction of contributors of that age that are still active).
